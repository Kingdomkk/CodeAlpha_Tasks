2. Phishing Awareness Training
📖 Description

An educational module designed to teach users how to detect phishing attempts and avoid cyber threats.

📚 Topics Covered
What is phishing?
Types of phishing attacks:
Email phishing
Spear phishing
Smishing (SMS phishing)
Social engineering techniques
🔍 How to Identify Phishing
Suspicious sender email address
Urgent or threatening language
Fake links or mismatched URLs
Requests for sensitive information
🛡️ Best Practices
Never click unknown links
Verify sender identity
Use multi-factor authentication (MFA)
Keep software updated
🧠 Interactive Ideas
Quiz questions
Real vs fake email comparisons
Scenario-based learning


3.Secure Coding Review
📖 Description

This project focuses on reviewing application code to identify security vulnerabilities and improve code safety.

🎯 Objectives
Identify common vulnerabilities
Recommend secure coding practices
Document findings
🔎 Common Vulnerabilities
SQL Injection
Cross-Site Scripting (XSS)
Hardcoded credentials
Insecure input validation
🛠️ Tools Used
Static analyzers (e.g., Bandit for Python)
Manual code review
